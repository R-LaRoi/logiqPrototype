# Replit Website Recreation Prompt

## Project Overview
Create a modern, professional website for Logiq Careers using the design aesthetic and layout structure from the accountant-128.webflow.io template, but populated with Logiq Careers' content and branding.

## Content to Include (from logiqcareers.com):

### Company Information
- **Company Name**: Logiq Careers
- **Tagline**: "Customized Career Services targeted toward those seeking more"
- **Mission**: Address unmet needs in career services with comprehensive personalized career services for individuals and companies in an increasingly competitive economy

### Services Offered
1. **FREE RESUME REVIEW**
2. **Career Coaching Sessions** (60-90 minute comprehensive sessions)
3. **Resume Writing Services**
4. **Interview Coaching**
5. **Career Guidance from Recruiters**

### Key Features
- Three free revisions with each order
- Additional revisions available at graduated fees
- Digital services (no refunds due to digital nature)
- Comprehensive exploration of interests, talents, and desires
- Professional career counseling and guidance

## Design Requirements (inspired by accountant-128.webflow.io):

### Layout Structure
- Clean, modern professional design
- Hero section with compelling headline and call-to-action
- Services section with clear service cards/blocks
- About/Mission section
- Testimonials/Client satisfaction section
- Contact/Get Started section
- Professional color scheme (suggest blues, whites, and accent colors)

### Technical Specifications for Replit
```html
- Use HTML5, CSS3, and vanilla JavaScript
- Responsive design (mobile-first approach)
- Modern CSS Grid and Flexbox layouts
- Smooth scrolling and subtle animations
- Professional typography (Google Fonts)
- Contact form functionality
- Clean, semantic HTML structure
```
